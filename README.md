# 🏦 Simple Bank Management System (C++)

### 🎓 Project Overview
This is a comprehensive **Bank Management System** developed as part of my journey as a **Computer Science Student**. The project focuses on handling real-world data persistence using **File Handling** and managing complex data structures with **Vectors** and **Structs**.

### 🚀 Current Features (V1.0):
- **Full CRUD Operations:** Create, Read, Update, and Delete client records.
- **Data Persistence:** All data is saved and retrieved from a text file (`Claients.txt`).
- **Search Functionality:** Find any client instantly by their account number.
- **Advanced String Parsing:** Custom `Split` logic to handle data separation.
- **Formatted Console UI:** Professional table display using `iomanip`.

### 🛠️ Technical Skills Applied:
- **File Streams:** Using `fstream` for database simulation.
- **Data Containers:** Managing dynamic data with `std::vector`.
- **Clean Code:** Modular functions for high readability and maintenance.
